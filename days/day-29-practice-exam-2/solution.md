## Key Concepts to Remember

### Must-Know Facts

**Delta Lake**:
- VACUUM retention: 30 days default
- Time travel: VERSION AS OF or TIMESTAMP AS OF
- OPTIMIZE: Compacts small files
- Z-ORDER: Max 4 columns recommended

**Structured Streaming**:
- Output modes: append, complete, update
- Checkpoint required for fault tolerance
- Watermark for late data handling

**MERGE**:
- WHEN MATCHED THEN UPDATE/DELETE
- WHEN NOT MATCHED THEN INSERT
- WHEN NOT MATCHED BY SOURCE THEN UPDATE/DELETE

**Unity Catalog**:
- Hierarchy: Metastore â†’ Catalog â†’ Schema â†’ Table
- Permissions: SELECT, MODIFY, CREATE, USAGE
- GRANT syntax: GRANT privilege ON object TO principal

**Jobs**:
- Job clusters: Start with job, terminate after
- Task dependencies: Linear, fan-out, fan-in
- Idempotency: Same result when run multiple times
